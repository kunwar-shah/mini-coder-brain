## 📋 Image Generation Workflow

### Step 1: Copy Prompt to claude.ai
1. Go to https://claude.ai
2. Start new conversation
3. Paste prompt from above
4. Wait for image generation

### Step 2: Review & Iterate
If image needs adjustments:
```
"Great! Can you make these changes:
- Make the text larger
- Change blue to darker shade
- Add more spacing between elements"
```

### Step 3: Download & Save
1. Click download button on generated image
2. Save to `.development/marketing/assets/generated/`
3. Rename with descriptive name

### Step 4: Optional Editing
Use Figma/Photoshop/GIMP to:
- Add final text overlays
- Adjust colors slightly
- Export in multiple formats

## 💡 Pro Tips

### Getting Better Results
1. **Be specific**: Include exact dimensions, colors, mood
2. **Iterate**: Ask for adjustments if not perfect
3. **Multiple attempts**: Generate 2-3 versions, pick best
4. **Reference existing**: "Style similar to GitHub's readme images"

### When to Use Generated vs Real
- **Generated**: Logos, banners, abstract concepts, comparisons
- **Real screenshots**: Actual product functionality, UI demos
- **Combination**: Generated background + real screenshot overlay

### Consistency
- Use same color palette across all images
- Maintain consistent icon style
- Keep typography choices limited (2-3 fonts max)
- Unified mood (professional tech aesthetic)
