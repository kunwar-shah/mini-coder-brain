## ğŸ’¡ Pro Tips

### Getting Better Results
1. **Be specific**: Include exact dimensions, colors, mood
2. **Iterate**: Ask for adjustments if not perfect
3. **Multiple attempts**: Generate 2-3 versions, pick best
4. **Reference existing**: "Style similar to GitHub's readme images"

### When to Use Generated vs Real
- **Generated**: Logos, banners, abstract concepts, comparisons
- **Real screenshots**: Actual product functionality, UI demos
- **Combination**: Generated background + real screenshot overlay

### Consistency
- Use same color palette across all images
- Maintain consistent icon style
- Keep typography choices limited (2-3 fonts max)
- Unified mood (professional tech aesthetic)
